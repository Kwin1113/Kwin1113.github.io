VIM编辑器

2019-02-04 Linux►常用命令 Linux 0 评论

VI编辑器的操作模式

  • Command Mode - 命令模式
  • Insert Mode - 输入模式
  • Last Line Mode - 底行模式

    常用指令

    底行模式常用指令

    :w
    :q
    :!
    :ls
    :n
    :15
    /xxx
    ?xxx

    命令模式常用指令

    h：光标左移
    j：光标下移
    k：光标上移
    l：光标右移
    ctrl+f：向下翻页
    ctrl+b：向上翻页
    ctrl+d：向下翻半页
    ctrl+u：向上翻半夜
    dd：删除光标所在行
    o：在光标所在行的下方插入一行并切换到输出模式
    yy：复制光标所在的行
    p：在光标所在行的下方粘贴
    P：在光标所在行的上方粘贴
